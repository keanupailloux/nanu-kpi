Google Apps Script setup guide
Follow these steps once to activate automatic email sync
›
1
Go to script.google.com and click New Project. Name it NANU Email Sync.
2
Delete everything in the editor and paste the entire script below.

        


        

          
3

          
On line 2 of the script, replace YOUR_GITHUB_PAGES_URL with your actual dashboard URL — for example https://keanupailloux.github.io/nanu-kpi/

        


        

          
4

          
Click Run → select checkOfferGunEmails → authorize when Google asks. This is a one-time permission for the script to read your Gmail.

        


        

          
5

          
Click Triggers (clock icon on the left) → Add Trigger → set it to run checkOfferGunEmails every 5 minutes. Click Save.

        


        

          
6

          
Click Deploy → New deployment → type: Web App → Execute as: Me → Who has access: Anyone → Deploy. Copy the Web App URL it gives you.

        


        

          
7

          
Paste your Web App URL below and click Save. The hot sheet will now auto-populate every time you open this page.
